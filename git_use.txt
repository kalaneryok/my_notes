6.查看远程仓库
	git remote
	0.github上面的远程仓库在创建的时候不要旋转readme的那个√，这个表示马上初始化，那么你本地的
	仓库需要上传过来就会提示报错，所以不要初始化
	1.将本地的仓库传送到远程仓库
	$ git remote add origin xxxxxx远程仓库地址
	2.推送到远端仓库
		git remote add origin git@github.com:kalaneryok/my_notes.git
		git push -u origin master
		后续不带u就行 -u关联分支
